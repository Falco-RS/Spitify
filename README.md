# Spitify
Proyecto II De Sistemas Operativos 

Requisitos del sistema
- Python 3.10+
- VLC Media Player 64-bit (necesario para python-vlc)
- FFmpeg en el PATH del sistema (conversión)
python-vlc es un binding que usa las DLLs/soporta de VLC. Si VLC no está instalado (o no es 64-bit), la reproducción fallará.

Instalación
1. Crear y activar entorno:
     python -m venv .venv
     .\.venv\Scripts\Activate.ps1
2. Dependencias Python:
     pip install -r requirements.txt
3. Instalar VLC y FFmpeg
     Buscar en el navegador VLC he instalar el de 64 bit
     FFmpeg: con Chocolatey (admin): choco install ffmpeg -y
4. Ejecutar
